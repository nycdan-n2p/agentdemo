# Как видеть обновления после правок в коде

При использовании **npm run start** сервер отдаёт уже **собранные** файлы. Пока ты не пересоберёшь проект, правки в коде не появятся на странице.

---

## Что делать после каждого изменения в коде

1. Останови сервер: в терминале, где он запущен, нажми **Ctrl+C**.

2. В папке проекта выполни **одну** команду:
   ```bash
   npm run refresh
   ```
   (Она сделает сборку и снова запустит сервер.)

3. Открой в браузере **http://localhost:3000** в режиме **инкогнито** (Cmd+Shift+N / Ctrl+Shift+N).

4. Обнови страницу (**F5** или **Cmd+R**). Должна открыться новая версия.

---

## Важно

- Адрес должен быть именно **http://localhost:3000** (не 127.0.0.1 и не другой порт).
- Режим инкогнито помогает избежать старого кэша.
- В проекте включены заголовки и мета-теги **no-cache**, чтобы при обновлении страницы браузер не подставлял старую версию из кэша.

# Как увидеть обновления (Google Sans)

1. **Останови старый сервер**  
   В терминале, где запущен `npm run start` или `npm run dev`, нажми **Ctrl+C**.

2. **Запусти сервер заново** (из папки проекта):
   ```bash
   cd "/Users/rpashkovskii/Downloads/agentdemo-main 3"
   npm run start
   ```

3. **Открой сайт без кэша:**
   - Запусти **режим инкогнито** (Chrome: Cmd+Shift+N / Ctrl+Shift+N).
   - В адресной строке введи: **http://localhost:3000**
   - Нажми Enter.

4. **Проверь, что загрузилась новая версия:**
   - Вверху страницы должна быть **зелёная полоска** с текстом про Google Sans.
   - Во вкладке браузера заголовок: **«Flex — Google Sans»**.

Если зелёной полоски и такого заголовка нет — открой именно http://localhost:3000 в инкогнито и сделай жёсткое обновление (Cmd+Shift+R).
